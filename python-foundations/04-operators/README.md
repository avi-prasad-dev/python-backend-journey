# Topic 4: Operators (Full Lesson)
## What is an Operator?
An operator is a symbol that performs an operation on values..

### Examples:
    10 + 5
    20 - 5
    10 * 2
    20 / 4

    Here +, -, *, /

is an operator

Without operators, software couldn't:
- Calculate salaries
- Compute taxes
- Process orders
- Compare passwords
- Validate user ages

## Types of Operators
## Arithmetic Operators
| Operator | Meaning        | Example   |
| -------- | -------------- | --------- |
| `+`      | Addition       | `10 + 5`  |
| `-`      | Subtraction    | `10 - 5`  |
| `*`      | Multiplication | `10 * 5`  |
| `/`      | Division       | `10 / 5`  |
| `%`      | Modulus        | `10 % 3`  |
| `**`     | Power          | `2 ** 3`  |
| `//`     | Floor Division | `10 // 3` |


## Modulus Operator %
Returns the remainder

    print(10 % 3)

Output:

    1

Industry Use:
- Checking even/odd number
- Pagination
- Cyclic operations

## Power Operator **

    print(2 ** 3)

Output:

    8

Meaning:
    2 x 2 x 2

## Floor Division //

    print(10 // 3)

Output:

    3

Removes decimal portion.


## Comparison Operators
Used to compare values.

| Operator | Meaning                | Example   |
| -------- | --------------         | --------- |
| `==`     | Equal                  | `10 == 5` |
| `!=`     | Not Equal              | `10 != 5` |
| `>`      | Greater Than           | `10 > 5`  |
| `<`      | Less Than              | `10 < 5`  |
| `>=`     | Greater Than or Equal  | `10 % 3`  |
| `<=`     | Less Than or Equal     | `2 ** 3`  |


## Logical Operators
    and
